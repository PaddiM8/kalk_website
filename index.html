kalk

A command line calculator that supports user-defined variables, functions, and to some extent units.

>> f(x, y) = Σ(1, 3, (2sin4/x!)^y % n) + cos(60°)

>> a = 3

>> f(a, 2) / 8

0.0863645842

>> Γ(⌊3π+2⌋ + √2)

1.1060309421365242*10^8

Features

  • Operators: +, -, *, /, !, %
  • Groups: (), ⌈⌉, ⌋⌊
  • Pre-defined functions and constants
  • User-defined functions and variables
  • User-defined units (experimental)
  • Understands fairly ambiguous syntax. Eg. 2sin50 + 2xy
  • Syntax highlighting
  • Completion for special symbols on tab
  • Sum functions
  • Load a file including predefined functions and constants
